`https://127.0.0.1:8000/code`

`generate-promo-codes [-a|--alphanumeric] [-h|--help] [-q|--quiet] [-v|vv|vvv|--verbose] [-V|--version] [--ansi] [--no-ansi] [-n|--no-interaction] [-e|--env ENV] [--no-debug] [--] <command> <length> <amount>`
